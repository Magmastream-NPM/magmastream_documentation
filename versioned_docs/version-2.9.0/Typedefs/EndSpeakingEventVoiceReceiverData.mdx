> Represents the data payload emitted when a user stops speaking in a voice channel.
> 
> | <i className="fa-solid fa-hashtag"/> Property | <i className="fa-solid fa-font"/> Type                                                            | <i className="fa-solid fa-list"/> Description                                                        | <i className="fa-solid fa-flask"/> Optional |
> |:----------------------------------------------|:--------------------------------------------------------------------------------------------------|:-----------------------------------------------------------------------------------------------------|:-------------------------------------------:|
> | `userId`                                      | [string](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/String) | ID of the user who stopped speaking.                                                                 | <i className="fa-solid fa-xmark"/>          |
> | `guildId`                                     | [string](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/String) | ID of the guild where the event occurred.                                                            | <i className="fa-solid fa-xmark"/>          |
> | `data`                                        | [string](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/String) | Base64-encoded audio stream data captured from the user.                                             | <i className="fa-solid fa-xmark"/>          |
> | `type`                                        | "opus" \| "pcm"                                                                              | Format of the audio data. Some older versions may also return `"ogg/opus"` as an unofficial variant. | <i className="fa-solid fa-xmark"/>          |
