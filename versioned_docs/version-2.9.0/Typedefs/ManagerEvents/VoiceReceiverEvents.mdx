# Voice Receiver Events

:::warning
⚠️ These events are only emitted by a [NodeLink <i className="fa-solid fa-arrow-up-right-from-square" style={{ fontSize: '0.8em'}}/>](https://github.com/PerformanC/NodeLink) node.
:::

> These events are fired when the NodeLink node emits its related events.

### VoiceReceiverConnect

> Emitted when a player's [voice receiver <i className="fa-solid fa-arrow-up-right-from-square" style={{ fontSize: '0.8em'}}/>](https://github.com/PerformanC/NodeLink/blob/main/docs/API.md#voice-receive) connects.
