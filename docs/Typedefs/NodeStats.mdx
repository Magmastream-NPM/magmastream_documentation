> Lavalink periodically sends detailed statistics about the node's performance and resource usage.  
> These stats are crucial for monitoring system load, audio frame issues, and overall node health.

---

### `NodeStats`

> Main structure containing node-wide statistics.
> 
> | <i className="fa-solid fa-hashtag"/> Property | <i className="fa-solid fa-font"/> Type                                                            | <i className="fa-solid fa-list"/> Description               | <i className="fa-solid fa-flask"/> Optional  |
> |:----------------------------------------------|:--------------------------------------------------------------------------------------------------|:----------------------------------------------------------- |:--------------------------------------------:|
> | `players`                                     | [number](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Number) | Number of connected players (active + idle)                 | <i className="fa-solid fa-xmark"/>           |
> | `playingPlayers`                              | [number](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Number) | Number of currently playing players                         | <i className="fa-solid fa-xmark"/>           |
> | `uptime`                                      | [number](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Number) | Node uptime in milliseconds                                 | <i className="fa-solid fa-xmark"/>           |
> | `memory`                                      | [`MemoryStats`](#memorystats)                                                                     | Memory usage statistics                                     | <i className="fa-solid fa-xmark"/>           |
> | `cpu`                                         | [`CPUStats`](#cpustats)                                                                           | CPU usage statistics                                        | <i className="fa-solid fa-xmark"/>           |
> | `frameStats`                                  | [`FrameStats`](#framestats) \| `null`                                                             | Frame loss and timing information (may be null if disabled) | <i className="fa-solid fa-check"/>           |

### `FrameStats`

> | <i className="fa-solid fa-hashtag"/> Property | <i className="fa-solid fa-font"/> Type                                                            | <i className="fa-solid fa-list"/> Description          | <i className="fa-solid fa-flask"/> Optional  |
