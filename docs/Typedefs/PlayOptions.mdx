> Represents the options for customizing Lavalink's playback behavior.
> 
> | <i className="fa-solid fa-hashtag"/> Property | <i className="fa-solid fa-font"/> Type                                                              | <i className="fa-solid fa-list"/> Description                                        | <i className="fa-solid fa-flask"/> Optional |
> |:----------------------------------------------|:----------------------------------------------------------------------------------------------------|:-------------------------------------------------------------------------------------|:-------------------------------------------:|
> | `startTime`                                   | [number](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Number)   | Start position in milliseconds. The track will begin playing from this timestamp.    | <i className="fa-solid fa-check"/>          |
> | `endTime`                                     | [number](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Number)   | End position in milliseconds. The track will stop at this point.                     | <i className="fa-solid fa-check"/>          |
> | `noReplace`                                   | [boolean](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Boolean) | If `true`, the new play payload will be ignored if another track is already playing. | <i className="fa-solid fa-check"/>          |
