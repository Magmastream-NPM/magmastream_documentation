# Band

> Band is object containing items from [`BandOptions{}`](#bandoptions)

## BandOptions\{}

> Represents an equalizer band.
>
> | <i className="fa-solid fa-hashtag"/> Parameter | <i className="fa-solid fa-font"/> Type                                                            | <i className="fa-solid fa-list"/> Description       |
> | :--------------------------------------------- | :------------------------------------------------------------------------------------------------ | :-------------------------------------------------- |
> | `band`                                         | [number](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Number) | The index of the equalizer band (0-12).             |
> | `gain`                                         | [number](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Number) | The gain value of the equalizer band (in decibels). |
