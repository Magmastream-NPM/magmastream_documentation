## unresolvedTrack{}

> Unresolved tracks can't be played normally, they will resolve before playing into a Track.

| <i className="fa-solid fa-hashtag"/> Parameter | <i className="fa-solid fa-font"/> Type                                                                                                                                                                                                                                                                         | <i className="fa-solid fa-list"/> Description                                |
| :--------------------------------------------- | :------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | :--------------------------------------------------------------------------- |
| `title`                                        | [string](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/String)                                                                                                                                                                                                              | The title to search against.                                                 |
| `author`                                       | [string](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/String)                                                                                                                                                                                                              | The author to search against.                                                |
| `duration`                                     | [number](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Number)                                                                                                                                                                                                              | The duration to search within 1500 milliseconds of the results from YouTube. |
| `resolve()`                                    | [function](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Function) returns [Promise](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Promise)[\<`void`>](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Operators/void) | Resolves into a Track.                                                       |
