### play() {#play}

> Plays the next track.<br/><br/>
> If a track is provided, it will be played. Otherwise, the next track in the queue will be played.<br/>
> If the queue is not empty, but the current track has not finished yet, it will be replaced with the provided track.
>
> <i className="fa-solid fa-right-to-bracket"/> Return: [Promise](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Promise)[\<void>](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Operators/void)
